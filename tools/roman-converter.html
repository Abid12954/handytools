Roman Numeral Rules

  • I = 1, V = 5, X = 10, L = 50, C = 100, D = 500, M = 1000
  • Letters can be repeated up to 3 times
  • When a smaller numeral appears before a larger one, subtract it
  • Only I, X, and C can be used for subtraction
